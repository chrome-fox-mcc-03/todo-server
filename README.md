# todo-server

## **Create New Todo**
Add a new todo list

* **URL**

  _/todos/_

* **Method**

  `POST`

* **Headers**
  **Required:**

  `token`

* **URL Params**
  **Required:**

  None

* **Data Params**

  title,
  description,
  status,
  due_date

* **Success Response:**
  * **Code:** 201
  * **Content:** 
  ```javascript
  {
    "Todo": {
        "id": 21,
        "title": "go home",
        "description": "go home at 6pm",
        "status": false,
        "due_date": "2020-03-02T10:44:12.905Z",
        "updatedAt": "2020-03-02T10:44:13.051Z",
        "createdAt": "2020-03-02T10:44:13.051Z"
    }
  }
  ```

* **Error Response:**
  * **Code:** 400
  * **Content:** 
  ```javascript
  {
    "error": "validation error" 
  }
  ```

    OR

* **Error Response:**
  * **Code:** 500
  * **Content:** 
  ```javascript
  {
    "error": "internal server error" 
  }
  ```

## **Find One Todo List by Id**

find one row from todo list by id 

* **URL**

  _/todos/:id_

* **Method**

  `GET`

* **Headers**
  **Required:**

  `token`

* **URL Params**
  **Required:**
  
  `id=[integer]`

* **Data Params**

   None

* **Success Response:**
  * **Code:** 200
  * **Content:** 
  ```javascript
  {
    "Todo": {
        "id": 21,
        "title": "go home",
        "description": "go home at 6pm",
        "status": false,
        "due_date": "2020-03-02T10:44:12.905Z",
        "updatedAt": "2020-03-02T10:44:13.051Z",
        "createdAt": "2020-03-02T10:44:13.051Z"
    }
  }
  ```

* **Error Response:**
  * **Code:** 404
  * **Content:** 

## **Create Countdown**

Create a countdown of todo's due date 

* **URL**

  _https://countdownmail.com/api_

* **Method**

  `POST`

* **Headers**
  **Required:**

  `Authorization=[string]`

* **URL Params**
  **Required:**
  
  None

* **Data Params**

  None

* **Success Response:**
  * **Code:** 200
  * **Content:** 
  ```javascript
  {
    "status": "success",
    "message": {
        "code": "1j",
        "src": "http://i.countdownmail.com/td.gif"
    }
  }
  ```

* **Error Response:**
  * **Code:** 401
  * **Content:** 
  ```javascript
  {
    "status": "error",
    "message": "Unauthorized"
  }
  ```
